 React Blog Application

📌 Project Overview

This project is a **React Blog Application** built using modern React concepts such as **functional components, React Hooks, and React Router**. The application allows users to create, view, and navigate blog posts through a clean and simple interface.

The main goal of this project is to demonstrate how **component-based architecture** works in React and how routing can be used to build multi-page applications without reloading the browser.

This project is suitable for beginners who want to understand how React applications are structured and how different components interact with each other.


🚀 Features

* View all blog posts on the Home page
* Create new blog posts
* Navigate between different pages using React Router
* View individual blog post details
* Responsive and simple user interface
* Component-based architecture
* Dynamic routing for blog posts
* Error handling for unknown routes (Missing Page)


🛠️ Technologies Used

* React.js
* JavaScript (ES6+)
* React Router DOM
* HTML5
* CSS3


📂 Project Structure

The application is divided into multiple reusable components to keep the code organized and maintainable.

Main Components include:

* Header
* Navigation Bar
* Home Page
* New Post Page
* Post Page
* About Page
* Missing Page (404)
* Footer

Each component is responsible for rendering a specific part of the user interface.


⚙️ How It Works

The application uses **React Router** to handle navigation between pages. Instead of loading a new page from the server, React dynamically renders the required component based on the URL.

React Hooks such as **useState** and **useEffect** are used to manage application state and handle side effects like fetching or updating data.


💡 Learning Objectives

This project helps in understanding:

* React component structure
* State management using Hooks
* Client-side routing using React Router
* Handling dynamic routes
* Organizing a React project for scalability


📦 Installation :

  Clone the repository
    
  git clone https://github.com/MeenalKesarkar/BlogApp-using-React
  
  Navigate to the project folder
  
  cd react-blog-app
  
  Install dependencies
  
  npm install
  
  Start the development server
  
  npm run dev
  
  The application will run on:
  
  http://localhost:5173
